说明

.ui-table提供一两个常用表格的样式
html 片段： 
<table class="ui-table ui-table-noborder">
    <tr>
        <th>#</th>
        <th>name</th>
        <th>age</th>
        <th>job</th>
    </tr>
    <tr>
        <td>1</td>
        <td>大宝</td>
        <td>53</td>
        <td>CEO</td>
    </tr>
    <tr>
        <td>2</td>
        <td>二宝</td>
        <td>50</td>
        <td>UFO</td>
    </tr>
    <tr>
        <td>3</td>
        <td>三宝</td>
        <td>47</td>
        <td>CTO</td>
    </tr>
</table>
