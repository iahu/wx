下拉列表样式，可以替换select

有二个可以配置的 class ：

  • .ui=dropdown-active：显示与隐藏
  • .ui=dropdown-border：下拉列表有没有分割线
html 片段： 
<div class="ui-dropdown ui-dropdown-border">
  <span class="ui-dropdown-show">
    <span>要显示的内容</span>
    <em>▼</em>
  </span>

  <div class="ui-dropdown-list-box">
    <ul class="ui-dropdown-list">
      <li>
        <em>♪</em>
        <span>选择一</span>
        <input type="radio" name="test_name_value" value="1">
      </li>
      <li>
        <em>♪</em>
        <span>选择二</span>
        <input type="radio" name="test_name_value" value="2">
      </li>
    </ul>
  </div>
</div>
