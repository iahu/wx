tab切换样式

.ui-tab-list > li 应与 .ui-tab-panel 对应

html 片段： 
<div class="ui-tab">
    <div class="ui-tab-hd">
        <ul class="ui-tab-list" role="tablist">
            <li class="ui-tab-active" role="tab"><a href="axtab1.html">标签1</a></li>
            <li role="tab"><a href="axtab2.html">标签2</a></li>
            <li role="tab"><a href="axtab3.html">标签3</a></li>
        </ul>
    </div>
    <div class="ui-tab-content" role="group">
        <div class="ui-tab-panel ui-tab-active" id="ui-tab-panel1" role="tab-panel" aria-hidden="false">
        </div>
        <div class="ui-tab-panel" id="ui-tab-panel2" role="tab-panel" aria-hidden="true">
        </div>
        <div class="ui-tab-panel" id="ui-tab-panel3" role="tab-panel" aria-hidden="true">
        </div>
    </div>
</div>
