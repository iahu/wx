说明

.tips 为基础样式 加上 .tips-error 等类型 class 可以设置不同类型的样式效果。corner 和 closer 部分是可选的

style属性的 left、top 可以用来定位

html 片段： 
<div class="tips" style="left:20px;top:20px;">
    <div class="tips-box tips-error">
        <div class="tips-corner">
            <i>♦</i>
            <em>♦</em>
        </div>
        <div class="tips-content">
            带关闭及方向指示角的提示啦。。。
        </div>
        <span class="tips-closer">×</span>
    </div>
</div>
