说明

弹出框的样式，与 panel非常像，不过它有个最小宽度420px

注意，设置高度的时候应该设置.pop-body的，而不是直接设置.pop-win

html 片段： 
<div class="pop-win" role="alertdialog">
    <div class="pop-header">
        <strong class="pop-title" aria-label="选择小区">李清照的辞</strong>
        <span class="pop-closer closer">×</span>
    </div>
    <div class="pop-body">
        <div class="pop-content">
        content...
        </div>
    </div>
</div>

一剪梅·红藕香残玉簟秋

红藕香残玉簟秋，轻解罗裳，独上兰舟。
云中谁寄锦书来？雁字回时，月满西楼。
花自飘零水自流，一种相思，两处闲愁。
此情无计可消除，才下眉头，却上心头。
